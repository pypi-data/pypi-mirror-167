# 祖先疗愈

## 下载

### Docker

```
docker pull apachecn0/zuxian-liaoyu
docker run -tid -p <port>:80 apachecn0/zuxian-liaoyu
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install zuxian-liaoyu
zuxian-liaoyu <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g zuxian-liaoyu
zuxian-liaoyu <port>
# 访问 http://localhost:{port} 查看文档
```
