# 魔法的启蒙

## 下载

### Docker

```
docker pull apachecn0/mofade-qimeng
docker run -tid -p <port>:80 apachecn0/mofade-qimeng
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install mofade-qimeng
mofade-qimeng <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g mofade-qimeng
mofade-qimeng <port>
# 访问 http://localhost:{port} 查看文档
```
