# 塔罗原理

## 下载

### Docker

```
docker pull apachecn0/taluo-yuanli
docker run -tid -p <port>:80 apachecn0/taluo-yuanli
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install taluo-yuanli
taluo-yuanli <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g taluo-yuanli
taluo-yuanli <port>
# 访问 http://localhost:{port} 查看文档
```
