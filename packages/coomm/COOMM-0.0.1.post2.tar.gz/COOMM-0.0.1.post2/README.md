## Project Description

Control Oriented Muscle Model for an octopus arm.

## Install

The package can be installed using standard python package installer (pip):

```bash
pip install coomm
```

or from source code:

```bash
cd <repository directory>
git clone https://github.com/hanson-hschang/COMM
pip install -e COMM
```
