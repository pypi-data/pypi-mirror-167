# 太阳回归

## 下载

### Docker

```
docker pull apachecn0/taiyang-huigui
docker run -tid -p <port>:80 apachecn0/taiyang-huigui
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install taiyang-huigui
taiyang-huigui <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g taiyang-huigui
taiyang-huigui <port>
# 访问 http://localhost:{port} 查看文档
```
