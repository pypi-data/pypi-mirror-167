## SQL ORDER BY 关键字:

SQL **ORDER BY** 关键字用于根据一列或多列按升序或降序对数据进行排序。

**语法:**

```java
SELECT column_name_1, column_name_2 FROM table_name ORDER BY column_name_1 ASC|DESC, column_name_2 ASC|DESC;
```

**举例:**

排序依据–升序:

```java
SELECT * FROM SCOREBOARD ORDER BY Playername ASC;
```
